# CV

## Pré-requis

**Installation de Node :**

  - Homebrew : `brew install node`
  - Manuellement : `http://nodejs.org/download/`

## Installation

    git clone https://github.com/LanF3usT/cv.git
    cd cv

**Lancer les commandes :**

    npm install
    npm install -g nodemon coffee-script mocha grunt-cli

## Démarrer le serveur

    npm start
